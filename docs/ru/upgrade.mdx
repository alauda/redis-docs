Служба кеширования Alauda для Redis OSS будет выполнять обновления на основе настроенной стратегии обновления:

- **Автоматически** : Автообновления запускаются немедленно при обнаружении новых версий компонентов.
- **Вручную** : Требуется ручное одобрение перед запуском процесса обновления.
