# Журнал

Понимание журналов операций экземпляра на уровне контейнера, таких как журналы ошибок Redis. Правильное использование журналов может помочь вам быстро выявлять проблемы и обрабатывать ошибки и исключения.

## Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите к **Redis** в левом навигационном меню.

    2. Выберите ***пространство имен***, содержащее ваш экземпляр.

    3. Нажмите на ***имя экземпляра***, чтобы получить доступ к его деталям.

    4. Откройте вкладку **实时日志** для просмотра потока живых журналов.

    5. Примените параметры фильтрации, чтобы сосредоточиться на соответствующей информации:
       - Выберите конкретный **Pod** из выпадающего меню, чтобы изолировать журналы от отдельных узлов
       - Введите поисковые термины для фильтрации записей журналов по содержимому
       - Отрегулируйте диапазон времени, чтобы сузить окно анализа

    6. Просмотрите отфильтрованные журналы, чтобы выявить шаблоны или аномалии, связанные с вашим расследованием.

    7. Для дальнейшего анализа или документирования нажмите кнопку экспорта <span style={{display: 'inline-block'}}>![Экспорт значка](../../en/assets/44buttonexport.png)</span>, чтобы загрузить данные журнала.
  </Tab>
</Tabs>
