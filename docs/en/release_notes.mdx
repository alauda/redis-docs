# Release Notes

## v4.0.0

### New and Optimized Features

#### Component-Platform Decoupling

Alauda Cache Services for Redis's lifecycle havs been decoupled from platform's, which means when customer needs a certain feature or a bug fix, Redis team can release immediately instead of waiting for the platform release together.

#### Real-Time Logs on Component Details Page

The logs on Redis details page have been replaced with real-time logs that update instantly, facilitating troubleshooting and issue analysis.
