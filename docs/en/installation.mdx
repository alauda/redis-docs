# Install

This guide outlines the installation process for Alauda Cache Service for Redis OSS, including its required dependencies. Follow these steps sequentially to ensure proper deployment.

## Installing Alauda Application Services Essentials

::: tip Optional
Alauda Application Services Essentials provides the core graphical management interface components. This installation can be skipped for environments using CLI-only administration.
:::

### Prerequisites

1. Obtain the Alauda Application Services Essentials installation package compatible with your platform version.
2. Ensure you have sufficient privileges to deploy cluster-level plugins in the `global` cluster namespace.

### Steps

<Tabs>
  <Tab label="Web Console">
    1. Navigate to **Marketplace** > **Cluster Plugins** in the platform's left navigation panel.
    2. Select the `global` cluster from the cluster selector.
    3. Locate **Alauda Application Services Essentials** in the plugin list and click **Install**.
  </Tab>
</Tabs>

## Installing Alauda Application Services RDS Framework

::: tip Optional
Alauda Application Services RDS Framework provides the database service management layer. This installation can be skipped for environments using CLI-only administration.
:::

### Prerequisites

1. Obtain the Alauda Application Services RDS Framework installation package compatible with your platform version.
2. Ensure you have sufficient privileges to deploy Operators in your target cluster.

### Steps

<Tabs>
  <Tab label="Web Console">
    1. Access the **Platform Management** interface.
    2. Navigate to **Marketplace** > **Operator Hub** in the left navigation panel.
    3. Locate **Alauda Application Services RDS Framework** and click **Install** to access the deployment configuration page.

    Configure the installation parameters as follows:

    | Parameter | Recommended Configuration |
    | :-------- | :------------------------ |
    | **Channel** | **stable** - The production-ready distribution channel |
    | **Version** | Latest available version (selected by default) |
    | **Installation Mode** | **Cluster** - Provides a single Operator instance that manages Redis resources across all namespaces, optimizing resource utilization |
    | **Installation Location** | **Recommended Namespace** - The system will automatically create the appropriate namespace if it doesn't exist |
    | **Upgrade Strategy** | **Manual** - Operator upgrades require explicit approval, providing controlled update management |

    4. Review the configuration settings in the installation dialog and click **Install** to deploy the RDS Framework Operator.
  </Tab>
</Tabs>

## Installing Alauda Cache Service for Redis OSS

### Prerequisites

1. Obtain the Alauda Cache Service for Redis OSS Operator installation package compatible with your platform version.
2. Ensure the Alauda Application Services RDS Framework has been successfully deployed in your target cluster.

### Steps

<Tabs>
  <Tab label="Web Console">
    1. Access the **Platform Management** interface.
    2. Navigate to **Marketplace** > **Operator Hub** in the left navigation panel.
    3. Locate **Alauda Cache Service for Redis OSS** and click **Install** to access the deployment configuration page.

    Configure the installation parameters as follows:

    | Parameter | Recommended Configuration |
    | :-------- | :------------------------ |
    | **Channel** | The appropriate channel is automatically selected based on your platform version |
    | **Version** | Latest available version (selected by default) |
    | **Installation Mode** | **Cluster** - Provides a single Operator instance that manages Redis resources across all namespaces, optimizing resource utilization |
    | **Installation Location** | **Recommended Namespace** - The system will automatically create the appropriate namespace if it doesn't exist |
    | **Upgrade Strategy** | **Manual** - Operator upgrades require explicit approval, providing controlled update management |

    4. Review the configuration settings in the installation dialog and click **Install** to deploy the Redis OSS Operator.
  </Tab>
</Tabs>
