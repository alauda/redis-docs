# 启停实例

一键启停功能满足日常开发运维活动需求，方便您快速暂停或开启中间件实例。在计划性停机、节点负载过大以及开发测试环境中的场景下，您可能需要临时释放 CPU、内存资源但不删除实例。一键启停功能帮助您节省资源并灵活管理中间件实例，从而降低成本并提高效率。


## 停止实例

### 前提条件

实例为 **处理中**、**运行中**、**异常** 或 **未知** 状态。

### 操作步骤

<Tabs>

<Tab label="Web Console">

**提示**：建议在停止实例前先做一次全量备份，防止数据丢失。

1. 在左侧导航栏中，单击 **Redis**。

2. 单击 ***命名空间名称***。

3. 单击 ***实例名称*** 。

4. 单击 **操作** > **停止**，并确认。
</Tab>
</Tabs>

### 停止后无法使用的功能

| 功能 | 说明 |
| --- | --- |
|**终端控制台**| 无法使用终端控制台。 |
|**操作**| 无法重启、更新、更新调度配置。 |
|**拓扑**| 无法查看拓扑页。 |
|**参数配置**| 无法更新参数和导出参数。 |
|**用户管理**| 无法查看用户管理页。 |
|**备份恢复**| 无法执行备份。 |
|**巡检**| 对当前实例的巡检结果无效。 |
|**访问方式**| 可能获取不到路由信息。 |

## 启动实例

### 前提条件

实例为 **停止中** 或 **已停止** 状态。

### 操作步骤

<Tabs>

<Tab label="Web Console">

1. 在左侧导航栏中，单击 **Redis**。

2. 单击 ***命名空间名称***。

3. 单击 ***实例名称*** 。

4. 单击 **操作** > **启动**，并确认。
</Tab>

</Tabs>
