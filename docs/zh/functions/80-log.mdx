# 日志

从容器层面了解实例运行过程中的日志，例如 Redis 错误日志。妥善使用日志可以帮助您快速定位问题，并处理故障和异常。

## 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航菜单中导航到 **Redis**。

    2. 选择包含您实例的 ***命名空间***。

    3. 点击 ***实例名称*** 以访问其详细信息。

    4. 打开 **实时日志** 标签以查看实时日志流。

    5. 应用过滤选项以专注于相关信息：
       - 从下拉菜单中选择特定的 **Pod**，以隔离单个节点的日志
       - 输入搜索词以通过内容过滤日志条目
       - 调整时间范围以缩小分析窗口

    6. 审阅过滤后的日志，以识别与您的调查相关的模式或异常。

    7. 为了进一步分析或文档用途，点击导出按钮 <span style={{display: 'inline-block'}}>![导出图标](../assets/44buttonexport.png)</span> 下载日志数据。
  </Tab>
</Tabs>
