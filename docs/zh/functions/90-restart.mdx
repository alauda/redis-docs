# 重启实例

实例运行过程中，遇到实例运行故障、连接数过多或其他无法定位原因的未知问题时，可以通过重启实例的操作进行问题修复。

## 注意事项

重启过程中将逐个重启 Pods，可能会造成业务的短暂中断。

## 前提条件

实例处于 **运行中** 状态。

**说明**：

- 如果实例处于 **处理中** 状态，说明实例可能正在正常启动，请耐心等待。

- 如果实例启动失败或异常，即实例处于 **处理中** 或 **错误** 状态。在健康检查超时后，实例将自动重启，请耐心等待。

## 操作步骤

<Tabs>

<Tab label="Web 控制台">

1. 在左侧导航栏中，单击 **Redis**。

2. 单击 ***命名空间名称***。

3. 单击 ***实例名称***。

4. 单击 **操作** > **重启**，并确认。

</Tab>
</Tabs>
