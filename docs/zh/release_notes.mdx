# 发布说明

## v4.0.0

### 新增和优化功能

#### 组件与平台解耦

灵雀云的 Redis OSS 缓存服务的生命周期已与平台解耦，这意味着当客户需要某个功能或修复错误时，Redis 团队可以立即发布，而不必等待平台的整体发布。

#### 组件详情页面的实时日志

Redis 详情页面上的日志已被替换为实时日志，可即时更新，方便进行故障排除和问题分析。

#### 通过 LoadBalancer 进行 Redis 外部访问

现在支持通过 LoadBalancer 进行 Redis 的访问。在创建实例时，用户可以选择通过 LoadBalancer 类型的服务将服务暴露给外部。

与 NodePort 相比，LoadBalancer 提供了更好的高可用性、性能以及标准端口的导出。建议在生产环境中使用 LoadBalancer 方法。

##### 限制

使用 LoadBalancer 时，请确保集群有可用的外部地址池。我们建议使用 MetalLB 来为集群创建外部地址池。
