# 发布说明

## v4.0.0

### 新增和优化功能

#### 组件与平台解耦

Alauda Cache Services for Redis 的生命周期已与平台解耦，这意味着当客户需要某个功能或修复 bug 时，Redis 团队可以立即发布，而无需等待平台的共同发布。

#### 组件详细信息页面的实时日志

Redis 详细信息页面上的日志已被实时日志替代，实时更新，方便故障排除和问题分析。
