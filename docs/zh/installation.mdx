# 安装

## 安装 Alauda Application Services Essentials 插件

::: tip 可选
该插件是 Alauda Application Services 图形管理界面的基础组件。如果您仅使用 CLI，可以跳过此步骤。
:::

### 前提条件

1. 下载与您的平台匹配的 Alauda Application Services Essentials 安装包。
2. 使用平台的应用上架能力，将 Alauda Application Services Essentials 上架到 `global` 集群。

### 操作步骤

<Tabs>
  <Tab label="Web Console">
    1. 在左侧导航栏中，进入 **市场** > **集群插件**。
    2. 选择 `global` 集群。
    3. 单击 **Alauda Application Services Essentials** 插件右侧的操作按钮 > **安装**。
  </Tab>
</Tabs>

## 安装 Alauda Application Services RDS Framework 插件

::: tip 可选
Alauda Application Services RDS Framework 是 Alauda Application Services 图形管理界面所需的组件。如果您仅使用 CLI，可以跳过此步骤。
:::

### 前提条件

1. 下载与您的平台匹配的 Alauda Application Services RDS Framework 安装包。
2. 使用平台的应用上架能力，将 Alauda Application Services RDS Framework 上架到您希望使用 Alauda Application Services 图形功能的任何集群。

### 操作步骤

<Tabs>
  <Tab label="Web Console">
    1. 登录平台，进入 **平台管理** 页面。
    2. 在左侧导航栏中，选择 **市场** -> **操作员中心**，进入 **操作员中心** 页面。
    3. 找到 **Alauda Application Services RDS Framework**，点击 **安装**，进入部署页面。

    配置参数：

    | **参数**             | **推荐配置**                                                                                                                  |
    | :------------------ | :--------------------------------------------------------------------------------------------------------------------------- |
    | **频道**             | 默认频道为 **stable**。                                                                                                       |
    | **版本**             | 默认选择最新版本。                                                                                                           |
    | **安装模式**         | **集群**：集群下所有命名空间共用一个 Operator 来创建和管理实例，从而降低资源消耗。                                          |
    | **安装位置**         | 选择 **推荐命名空间**：如果不存在，将自动创建。                                                                              |
    | **升级策略**         | **手动**：当 Operator Hub 有新版本时，需要手动确认以升级 Operator 到最新版本。                                              |

    4. 在弹出的安装页面上，使用默认配置，然后点击 **安装** 完成 **Alauda Application Services RDS Framework** 的部署。
  </Tab>
</Tabs>
